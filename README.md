# Django Geo Project

A Django-based GIS application using GeoDjango.

## Tech Stack
- Python
- Django
- GeoDjango
- PostgreSQL + PostGIS
- GDAL

## Setup Instructions

```bash
git clone https://github.com/yourusername/django-geo-project.git
cd django-geo-project
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
